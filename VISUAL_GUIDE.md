## 📐 Layout Breakpoints

### Desktop (1024px+)
- Full width container (max 1280px)
- 3-column event cards
- Corner diyas visible
- Large rangoli (320px)
- Hover effects active

### Tablet (768px - 1023px)
- Medium container (max 768px)
- 3-column event cards (smaller)
- Corner diyas hidden
- Medium rangoli (240px)
- Touch-friendly buttons

### Mobile (< 768px)
- Full width (padded)
- Single column
- No corner diyas
- Small rangoli (200px)
- Large touch targets
